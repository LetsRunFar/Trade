# Trade

> vue + webpack + express 移动端项目，支持node本地服务器测试接口

## 运行步骤

``` bash
# 安装
npm install
cd backEnd
npm install

# 运行前端项目（热更新）
npm run dev

# 运行node服务器（测试接口 + 在线聊天）
cd backEnd
npm test

# 页面打开

主页面：http://localhost

聊天页面：http://localhost:3000/client.html?cid=xxx&sid=xxx  http://localhost:3000/service.html?cid=xxx&sid=xxx

# 打包
npm run build
```
